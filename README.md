# SIIP: Serverless Intelligent Insights Pipeline

Proyecto de arquitectura serverless en AWS para la ingesta, 
procesamiento con IA y visualización de datos no estructurados.

## Stack Tecnológico
* **IaC:** Terraform
* **Backend:** AWS Lambda (Python), Step Functions, DynamoDB
* **Frontend:** React + Tailwind CSS hosted on S3
* **AI:** Rekognition, Transcribe, Translate, Comprehend

---

## English

# SIIP: Serverless Intelligent Insights Pipeline

serverless architecture project on AWS for ingestion, AI processing, and visualization of unstructured data.

## Technology Stack
* **IaC:** Terraform
* **Backend:** AWS Lambda (Python), Step Functions, DynamoDB
* **Frontend:** React + Tailwind CSS hosted on S3
* **AI:** Rekognition, Transcribe, Translate, Comprehend
